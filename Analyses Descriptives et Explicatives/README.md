## Description

Projet portant sur l'analyse de données Open Food Facts.

Une partie du processus de nettoyage des données se trouve dans le notebook [Nettoyage Base de Données](https://github.com/SanaeSaccomano/Data-Science/blob/main/Analyses%20Descriptives%20et%20Explicatives/Nettoyage%20Base%20de%20Donn%C3%A9es.ipynb)

L'analyse de la base de données se trouve dans le notebook [Analyse Descriptive et Explicative](https://github.com/SanaeSaccomano/Data-Science/blob/main/Analyses%20Descriptives%20et%20Explicatives/Analyse%20Descriptive%20et%20Explicative.ipynb)

Quelques visualisations statiques et intéractives se trouvent dans le notebook [C'est Beau et Ca Bouge](https://github.com/SanaeSaccomano/Data-Science/blob/main/Analyses%20Descriptives%20et%20Explicatives/C'est%20Beau%20et%20Ca%20Bouge.ipynb)

Un coup d'oeil au projet se trouve au sein de la présentation [C'est du Bon](https://github.com/SanaeSaccomano/Data-Science/blob/main/Analyses%20Descriptives%20et%20Explicatives/C'est%20du%20Bon.pdf)
 
## Compétences
- Nettoyage de données
- Visualisation intéractive de données
- Analyses Descriptives (Corrélation de Pearson) et Réduction de dimensions (PCA)
- Analyses Explicatives (ANOVA, CHI-2)

## Librairies

- numpy
- matplotlib
- seaborn
- datetime
- sklearn
- scipy
- statsmodels
- ipywidgets
- plotly

*Note : les fonctions personnelles de ce projet ne sont pas explicitées pour des raisons de confidentialité mais sont expliquées dès leur premier appel dans le script.*
